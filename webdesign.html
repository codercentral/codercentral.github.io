Introduction to Web Design
Everything is on the internet now in one way or another. But there just aren’t enough web designers! Currently professional web designers can earn up to thousands of dollars per website due to the lack of web designers. This course will unleash your creativity and inspire you to make a truly incredible website! During the course, students will be doing freelance work for local small businesses to help spice up their websites as well as adding to students’ personal projects.

Course Overview

1 Reading and understanding HTML and CSS
2 Basic HTML: tags, attributes, linking
3 Basic CSS: Selecting, properties, and values
4 Group projects with Git
5 Understanding and using Bootstrap
6 Responsive Web Design
7 Basic understanding of the internet
8 Working with graphic designers
9 Build website for local small business
10 Showcasing final website
